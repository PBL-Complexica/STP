# Venv Activation Instructions
Note: do it from ./STP/src/back/

## 0 Create venv
```bash
python3 -m venv /app_venv/
```

## 1.1 Windows PowerShell:
```bash
./app_venv/Source/activate.ps1
```

## 1.2 Bash (linux):
```bash
source ./app_vevn/Source/activate
```

## 2 Requirements:
```bash
pip install -r requirements.txt
```
